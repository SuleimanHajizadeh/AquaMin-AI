### 🔧 Sadə nümunə kod (nasosu idarə etmək üçün)

Bu kod motoru yandırıb-söndürəcək:

```cpp
int motorPin = 8;

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  digitalWrite(motorPin, HIGH); // ON
  delay(3000);

  digitalWrite(motorPin, LOW);  // OFF
  delay(3000);
}
```

### 🧠 Vacib məqam (indidən bilməlisən)

👉 **tranzistor vasitəsilə nasosu idarə edəcək**

```
Arduino pin → tranzistor → motor → batareya
```

## ✅ DOĞRU sensor (çox vacib)

👉 **DS18B20**

Niyə?

* Suya salınır ✔
* Dəqiqdir ✔
* AI üçün real data verir ✔

---

## 🔥 Sənə real tövsiyə (indi mağazadasan deyə)

👉 Əgər seçim varsa:

* **DHT11 qoy yerinə**
* **DS18B20 götür**

👉 Əgər DS18B20 yoxdursa:

* DHT11 al (boş getmə)
* amma sonra mütləq dəyişəcəksən

---

## 💡 Qısa qərar

* DHT11 → ⚠️ köməkçi, amma əsas deyil
* DS18B20 → ✅ əsas sensor

---

İstəsən, sənə:
👉 DHT11 ilə belə **AI demo sistemi necə qurmaq olar**
və ya
👉 DS18B20 ilə **real işlək sistem**

ikisini də addım-addım qurum.
Şəkilə baxanda səndə artıq bunlar var:

* Breadboard (maket plata)
* Jumper kabellər (çox yaxşı 👍)
* Pin header (qara çıxıntılı hissə)
* Bir neçə əlavə kabel

👉 Yəni **hardware başlanğıcı hazırdır**, indi sən düz düşünmüsən: *software hissəni hazırlamaq vaxtıdır.*

---

## 💻 İndi konkret nə etməlisən (software tərəf)

### 1. Arduino IDE qur (birinci və əsas addım)

Əgər hələ qurmamısansa:

* Arduino IDE yüklə və install et
* Aç və işlədiyini yoxla

---

### 2. İlk test kodunu hazırla (Blink)

Bu sənin “hello world”-undur 👇

```cpp
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
```

👉 Bunu indi yaz və saxla
Adını qoy: `test_blink.ino`

---

### 3. Struktur düşün (çox vacib mərhələ)

Sən layihə qurursan (sadəcə LED yox), ona görə əvvəlcədən plan qur:

### Sənin gələcək sistemin belə olacaq:

* Sensor → məlumat verir
* Arduino → qərar verir
* Çıxış (relay/pompa) → icra edir

👉 Software-də bunu belə düşün:

```cpp
oxuSensor();
analizEt();
qerarVer();
icraEt();
```

---

### 4. Sadə “control logic” yaz (hələ hardware olmasa da)

Məsələn, gələcək nasos üçün:

```cpp
int motorPin = 8;

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  // TEST LOGIC (sonra sensorla dəyişəcək)
  digitalWrite(motorPin, HIGH);
  delay(2000);

  digitalWrite(motorPin, LOW);
  delay(2000);
}
```

👉 Bu kod:

* gələcəkdə pompanı idarə edəcək
* indi sadəcə test üçündür

---

### 5. Serial Monitor istifadə etməyi öyrən

Bu çox vacibdir (debug üçün):

```cpp
void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.println("System is running...");
  delay(1000);
}
```

👉 Bu sənə:

* sistemin işlədiyini göstərir
* gələcəkdə sensor datanı izləməyə kömək edəcək
